# Chaos-Rush-Mobile
Jogo mobile, inspirado em Chaos Rush, encontrado no link https://chaos-rush2.onrender.com. Jogo do Maurício, que será feito em sua versão mobile com mais 4 desenvolvedores.

Para jogar, vc deve ter o Live Server instalado no VsCode, e após ter clonado o projeto, rode 'npm install' no terminal. Por fim, clique em 'Go Live' para jogar localmente. 

Estamos abertos à críticas e feedbacks!
